Feature Selection 
=================
The data in this datset is taken from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
These data are said to be from the Samsung Galaxy S smartphone originally stored at http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones 

The set of variables that were calculated from the original signal data signals are: 
====================================================================================
mean(): Mean value
std(): Standard deviation
meanFreq(): Weighted average of the frequency components to obtain a mean frequency


The following is a list of all of the feature names
===================================================
Subject	: The number id of the subject. It is an integer value in the range of 1 to 30.			
Activity : The activity the subject was doing. It is in the set {"LAYING","SITTING","STANDING","WALKING","WALKING_DOWNSTAIRS","WALKING_UPSTAIRS" }
